# 🛒 Ecommerce Frontend Design

A modern and responsive **eCommerce frontend website** built as part of the DevelopersHub Internship task. This project focuses on implementing a **pixel-perfect desktop UI** using HTML, CSS, and JavaScript based on a Figma design.

---

## 📌 Project Overview

This project is divided into **3 structured phases (weeks)** to simulate real-world frontend development workflow:

### 🔹 Week 1: Layout & Structure

* Designed Header (Logo, Search Bar, Navigation)
* Created Footer with multiple sections
* Setup project folder structure
* Focused on spacing, alignment, and typography

### 🔹 Week 2: Home & Product Listing

* Hero section with call-to-action
* Featured categories section
* Product listing page using CSS Grid
* Interactive hover effects on cards & buttons

### 🔹 Week 3: Product Details & Interactivity

* Product details page (image, price, description)
* Size selection dropdown
* JavaScript-based search interaction
* Basic UI interactivity and testing

---

## 🖥️ Features

✨ Modern UI Design
